
# User Input and Type Conversion


##Introduction
 * User input allows a program to accept data from the user while it is running. In Python, the input() function is used to take input    from   the keyboard.
 * User Input Examples
   **Taking Name as Input
    name = input("Enter your name: ")
    print("Hello,", name)

 * Type conversion is the process of converting one data type into another. This is important because the input() function always returns data as a string.
 * Type Conversion Examples
   ** String to Integer
      age = int(input("Enter your age: "))
      print("Next year you will be", age + 1)
   
   ** String to Float
      height = float(input("Enter your height in meters: "))
      print("Height:", height)


# Key Learnings
* input() is used to take user input.
* User input is stored as a string by default.
* int() converts data to integer.
* float() converts data to decimal numbers.
* str() converts data to string.
* Type conversion is essential when performing calculations on user input.
* User input and type conversion are widely used in real-world Python applications and data analysis projects
